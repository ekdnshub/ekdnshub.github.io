파이썬 파일 읽기(Python File Read)

* 이 포스팅은 네이버 블로그에서 작성(2012.11.06)한 내용을 옮겨온 것입니다.

파이썬이라는 스크립트 언어가 있다. 인터프리터형식이라 테스트해보기도 훨씬 간편하고 좋긴 하지만 인덴트는 영 적응이 안된다 -_-. 자바와 C에 익숙해져 있기 때문 TT

오늘은 가볍게 읽기만 해봄.

# 파일 읽기 예제
>>> fp = open("D:\\python\\test.txt")
>>> for line in fp: 
	print(line) # test

되게 단순하지만, 파일 읽기는 이거부터 시작이니 여기서 다양한 응용이 가능하겠지 뭐...
