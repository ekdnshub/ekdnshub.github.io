Getting Started - Intasll Go

Install Go

기본적으로 Go를 사용하기 위해서 다운로드를 해야한다. 포스팅 작성 시점에서 Go 인스톨 방법이 있는 링크는 이곳이다.

Download the Go distribution이라는 항목 밑에 큼지막한 글씨로 Download Go 가 쓰여있고 클릭하라는 말도 있다. 클릭하면 다운로드 받을 수 있는 링크로 연결된다.

해당 링크에서 안정화 버전들Stable versions이 있는데 지금은 1.4.1이 가장 최신이다. 공부하고자 하는 OS로 나눠받을 수 있게 되어 있다. 여기서는 원하는것으로 받아서 각 OS에 맞게 설치하면 된다. 이것에 대해서는 여기에서 Install the Go tools를 참고하자.

Test Go

설치를 끝마쳤다면 정말 잘 설치 되었는지 간단한 프로그램을 통해 테스트 해보자. 다음은 hello.go 라는 이름의 파일이다. 똑같이 작성하자.

/* hello.go */
package main

import "fmt"

func main(){
	fmt.Printf("hello, world\n")
}

파일 저장을 끝낸 뒤에 다음과 같은 명령어를 실행해보자.

$ go run hello.go

실행해서 다음과 같은 문구가 나오면 Go 설치 및 테스트까지 완료한 것이다.

hello, world

환경 구성

어느정도 Go에 대해서 구성이 끝났지만 실제로 구동하는 환경에서 몇가지 속성을 더 추가시켜줘야 할 수도 있다. 그 경우에는 여기를 참조하자. 정말로 필수적인 환경 구축에 대한 소개가 적혀 있다.
