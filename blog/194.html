블로그에 이력 보기 버튼이 추가 되었습니다.

2015.11.19 깃헙으로 변경하면서 이력 추적이 쉬워졌습니다. 따라서 이력 버튼은 삭제되었습니다. TT

블로그 포스트 제목 바로 밑 추가 기능으로 "이력" 버튼이 추가 되었습니다. 생긴 모양은 요로코롬 생겼습니다. 이 버튼은 제가 기존의 포스트에 내용을 추가할 때 수정 날짜를 안보이게 적어넣는데 그것을 보이게끔 해주는 기능을 활성화 시킵니다. 뭔가 문맥이 시공간을 뛰어넘고 있다는 생각이 드시면 이 버튼을 클릭해서 포스트의 어떤 부분이 언제 수정된건지 확인해 보세요.

기능 버튼을 눌러 활성화를 했음에도 별다른 변화가 없다면 해당 포스트는 해당일에 작성된 내용만 있는 것입니다. :D

해당 기능을 확인해 볼 수 있는 포스트는 실시간 현재 접속자 위치 추가를 참고하세요!
