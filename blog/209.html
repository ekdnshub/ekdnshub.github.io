Mac Apache

간단하게 Mac X OS상에서 아파치Apache를 설치하는 법을 알아봅시다.

get root authority

아파치를 설치하기 전 터미널을 띄운 뒤에 다음과 같은 커맨드를 이용해 root 권한을 획득합니다.

$ sudo su -

Enable Apache on Mac OS X

그리고 apachectl 커맨드를 이용해 아파치를 구동합니다.

$ apachectl start

check your web browser

이후 http://localhost로 접속이 잘 되는지 확인합니다. 그리고 기본 도큐먼트 루트Document root는 /Library/WebServer/Documents입니다.

Disable Apache on Mac OS X

아파치를 내릴 때는 다음과 같은 커맨드를 이용합니다.

$ apachectl stop

apache config

아파치 설정은 다음 파일을 수정해줍니다.

$ vi /etc/apache2/httpd.conf

Tip

로컬호스트에서만 접근하고 싶다면 http.conf 설정 중에 Listen 127.0.0.1:80을 넣어주면 됩니다. (기존 Listen 구문이 있다면 변경하면 됩니다.)
