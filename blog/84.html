C 파일 사이즈 구하기(Getting file size with C)

파일 사이즈를 구하는것도 C에서 간단한 것은 아니군요. 다음처럼 이용합시다...

#include <stdio.h>
...
// 파일 사이즈 체크
string filePath = "...";
FILE *f;
f = fopen(filePath, "r");
fseek(f, 0, SEEK_END);
int fileLength = ftell(f);
fclose(f);
...

fileLength 값으로 사이즈 크기를 알 수 있습니다.
