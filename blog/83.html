Scala Hello World!

처음 프로그래밍 언어를 배울땐 Hello, world!를 찍는게 기본이었던 것 같네요. 그래서 스칼라로도 한번은 해봐야죠!

Hello, world!

/* helloworld.scala */
object helloworld {
    def main(args: Array[String]){
        println("Hello, world!") // Hello, world!을 출력
    }
}

익숙하지 않은 키워드가 있으니, 바로 object라는 키워드일겁니다. 객체라고 하는데 클래스의 유일한 인스턴스를 만들기 위해 사용합니다. 디자인 패턴 중 "싱글튼 패턴" 아시죠? 그겁니다. 기본 소스를 조금 바꿔볼까요. 이번엔 scala 객체 하나를 더 추가해 봅시다. 그리고 helloworld에서 해당 객체를 호출해 보죠.

object keyword를 사용한 helloworld

object helloworld {
    def main(args: Array[String]){
        println("Hello, world!")
        objectCls.hello()
    }
}

object objectCls {
    def hello(){
        println("this object name is objectCls.")
    }
}

Hello, world!
this object name is objectCls.

이렇게 하면 java class로 열심히 싱글튼 클래스로 만들 필요 없을 것 같네요. object 키워드 선언 딱 해주면 간편하게 싱글튼 객체가 생깁니다. 이건 편리하군요. 아무튼 이번 포스팅은 console에 출력만 해보면 되니까, 미션 클리어!
