스칼라 개발 툴 다운로드(IDE for Scala)

스칼라 공부를 시작하고자 합니다. 함수형 프로그래밍을 제대로 배워보고 싶지만... 그냥 끄적거리는 수준이 되지 않을까 하네요. 그전에 스칼라 공부를 도와줄 IDE부터 설치해보고자 합니다.

여러가지 IDE가 있는 것 같지만, 개인적으로 익숙한 이클립스를 베이스로 하는 IDE를 선택했어요. 다음 링크에서 다운로드 받을 수 있습니다. http://scala-ide.org/download/sdk.html

포스팅하는 시점에서는 4.0.0 릴리즈 버전이 있네요. 이걸 가지고 하나씩 테스트 해보는 공간을 가지려 합니다. :3

failed to create the java virtual machine error 처리

failed to create the java virtual machine error라는 오류가 생기면 이클립스 디렉토리에서 eclipse.ini 파일 내부에 다음처럼 입력해줍니다.

-vm
"C:\Program Files\Java\jdk1.7.0_71\bin\javaw.exe" <-- 이부분은 각자 경로에 맞게 수정!

디렉토리명에 빈칸이 있는 경우 반드시 더블 쿼테이션(")으로 묶어주세요.

으, 위처럼 경로를 추가해도 같은 오류가 생기는 경우에는 -Xmx 부분을 적절하게 낮춰줍니다. -Xmx512m 처럼요. 2015.03.10 추가
