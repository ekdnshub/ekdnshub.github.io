MariaDB - Multi In Query

요즘은 뭔가 바쁜 시기입니다. :3 간단하게 MariaDB에서 Multi "IN" Query를 사용하는 것을 알아봅시다.

When do I use?

개인적으로는 Mybatis에서 Multi Get 쿼리를 만들때 사용했습니다. 간단하게 예제만 보면 될것 같네요.

Example

SELECT col1, col2, ...
FROM tbl
WHERE
( col1, col2 ) IN (
	( 'col1_param', 'col2_param' )
);
