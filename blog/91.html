select query를 실행하는 순서가 문득 어떻게 되는지 궁금해졌습니다. 그냥 아~ 이런거구나 보단 문서로 확인하고 싶었어요. 지금 포스팅의 내용은 이곳에서 일부 인용 되었습니다.

select query ordering

FROM
ON
JOIN
WHERE
GROUP BY
CUBE | ROLLUP
HAVING
SELECT
DISTINCT
ORDER BY
TOP

select 쿼리는 위부터 아래의 순서로 실행이 됩니다. 자주 사용하는 구문은 두껍게 해놨습니다. :3
