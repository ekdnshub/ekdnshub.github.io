블로그의 새로운 기능 소개

제 블로그를 오랜만에 뜯어보니 유저들이 포스트 조회할 때 불편할 것 같아 기능을 추가해봤습니다.

목차 버튼 추가

포스트 제목 밑에 목차 버튼을 신설했습니다. 내용이 긴 포스트의 경우 필요한 부분을 바로 보이게 할 때 유용할 것 같아요. 클릭하면 해당 내용으로 이동합니다.

소제목 클릭시 URL 복사 (여기를 클릭하면 URL이 복사됩니다.)

공유하기 쉬우시라고 소제목제목은 안되요.을 클릭하면 클립보드에 URL이 복사됩니다. 공유 많이 해주세요.

코드 블록 클릭시 코드 복사

내용이 긴 코드의 경우 복사하기 위해 스크롤하는 것이 불편하셨을 것 같아서 코드 블록도 클릭하면 코드가 클립보드에 복사되도록 기능을 추가해뒀습니다.

void main() {
    System.out.println("이 곳을 클릭하면 코드가 복사됩니다!");
}

URL 복사나 코드 복사의 경우 토스트 메시지가 안나오면 그냥 안되나보다 하고 넘기시면 됩니다. (...)
