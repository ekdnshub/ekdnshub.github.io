신경망(Neural Network)

In machine learning and cognitive science, artificial neural networks (ANNs) are a family of statistical learning algorithms inspired by biological neural networks (the central nervous systems of animals, in particular the brain) and are used to estimate or approximate functions that can depend on a large number of inputs and are generally unknown. Artificial neural networks are generally presented as systems of interconnected "neurons" which can compute values from inputs, and are capable of machine learning as well as pattern recognition thanks to their adaptive nature. 2015.04.10

Wikipedia - Artificial neural network

기계 학습 및 인지 과학 분야에서 인공 신경망(ANNs)은 통계적인 학습 알고리즘의 구성원입니다. 이러한 알고리즘은 생물학의 신경망(동물의 중추 신경 시스템에서 특히 뇌)에서 영감을 받은 것입니다. 또한 알고리즘은 평가 또는 많은 수의 입력값과 일반적으로 알려지지 않은 것에 의존하는 대략적인 기능에 사용됩니다. 인공 신경망은 일반적으로 서로 연결된 "뉴런"들의 시스템으로써 표현 됩니다. 이 뉴런들은 입력으로부터 값을 계산할 수 있고 그들의 자연스러운 적응 덕택에 패턴인지 머신 러닝에 알맞습니다. 2015.04.10

Java Neural Network Framework Neuroph(Neuroph)

GUI 인터페이스가 돋보이는 라이브러리입니다. GUI 화면을 이용해 기반을 만들고 이것을 자바 코드로 변환하면 됩니다. 일단 소개 문구부터 볼까요.

Neuroph is lightweight Java neural network framework to develop common neural network architectures. It contains well designed, open source Java library with small number of basic classes which correspond to basic NN concepts. Also has nice GUI neural network editor to quickly create Java neural network components. It has been released as open source under the Apache 2.0 license, and it's FREE for you to use it.

Neuroph

경량 자바 신경망 프레임워크라고 합니다. GUI 신경망 에디터로 빠르게 자바 신경망 콤포넌트를 구축할 수 있다고 하네요. 그리고 무료라는 것에 강조를 하고 있습니다.

어찌되었든 GUI를 통해 먼저 기반을 닦은 후 그것을 자바 코드로 짜면 된다는 것이 좋아보이네요.

Fast Artificial Neural Network Library(FANN)

다양한 언어로 바인딩을 해주고 있는 것이 특징입니다. 소개 문구를 봅시다.

Fast Artificial Neural Network Library is a free open source neural network library, which implements multilayer artificial neural networks in C with support for both fully connected and sparsely connected networks. Cross-platform execution in both fixed and floating point are supported. It includes a framework for easy handling of training data sets. It is easy to use, versatile, well documented, and fast. Bindings to more than 20 programming languages are available. An easy to read introduction article and a reference manual accompanies the library with examples and recommendations on how to use the library. Several graphical user interfaces are also available for the library.

FANN

FANN은 프리 오픈 소스 신경망 라이브러리라고 합니다. 중간 부분쯤 FANN은 트레이닝 데이터 집합의 쉬운 처리를 위해 프레임워크를 포함한다고 합니다. 사용하기 쉽고 다목적으로 활용하고 문서화가 잘 되어 있고 빠르다고 하는 것이 눈에 띄네요. 게다가 20개의 프로그래밍 언어 바인딩을 지원한다고 합니다. 그래서 FANN을 자바로 바인딩 할 수 있는 프로젝트를 찾아봤습니다. 이름은 FANNJ라고 하네요.

이곳을 클릭하면 FANNJ를 올려둔 Gitgub 사이트로 연결됩니다. 다만 FANN을 자바로 활용하려면 우선적으로 C로 작성된 FANN 라이브러리를 설치를 해야 한다고 하네요. 순수하게 자바로만 구동하는 것을 원한다면 FANN은 조금 꺼려질 수 있을 것 같습니다.
