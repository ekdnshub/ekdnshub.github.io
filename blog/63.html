Grails란 무엇인가?

Grails라는 오픈 소스가 있습니다. 공식 사이트는 이곳입니다. 이 오픈 소스를 사용하고 있는 곳이 링크드인 외에도 몇 곳 있다고 하네요. 아무튼 공식 홈페이지의 인트로에 있는 내용입니다.

What is Grails?

Grails is a powerful web framework, for the Java platform aimed at multiplying developers’ productivity thanks to a Convention-over-Configuration, sensible defaults and opinionated APIs. It integrates smoothly with the JVM, allowing you to be immediately productive whilst providing powerful features, including integrated ORM, Domain-Specific Languages, runtime and compile-time meta-programming and Asynchronous programming.

https://grails.org

굵게 강조된 것만 봐본다면 Grails는 아주 강력한 웹 프레임워크군요. 그리고 통합에 대하여 대부분 설명하고 있습니다. Grails는 JVM과 자연스럽게 통합하며 통합 ORM, 도메인별 언어를 포함하여 강력한 기능을 제공하고 런타임과 컴파일 시점 메타 프로그래밍 그리고 비동기적 프로그래밍등이 강점이라고 합니다. 그 외에도 감각적인 기본 설정과 의미가 분명한 API들을 이용해서 생산적이라고 하네요.

의역이라 뜻이 제대로 전달 됐는지는 잘 모르겠습니다. :3 (어설픈 영독해)

간단히 말해서 Grails는 Spring같은 웹 애플리케이션 프레임워크입니다.

흥미롭네요. 요즘은 볼거리가 너무 많은 것 같아요.
