title=Postman - Rest API Test Tool created=2015-10-04

오늘은 크롬 애플리케이션 중 하나인 Postman에 대해 소개해 드립니다.

Postman?

REST API 테스트 또는 URL 응답 테스트나 그 외 HTTP 프로토콜을 위한 테스트를 위한 최적의 도구입니다. 한마디로 웹 개발자 또는 서버 API 개발자라면 한번쯤 써보는 것을 추천합니다.

Install

Postman을 설치하려면 우선 크롬 브라우저를 먼저 설치 하신 후 다음의 링크를 클릭해 애플리케이션을 설치합니다. Postman Application Link

Tip

Postman 설치 이후 [Settings] - [Theme] 에서 검은색 배경의 테마를 선택할 수 있습니다.

Settings에서 "Response Font Size" 항목의 값을 조절해서 응답 문자열의 크기를 지정할 수 있습니다.
