title=Sublime Text 2 created=2015-10-23

불과 얼마 전에 TextWrangler에 대해 포스팅을 했었는데 이번에도 Mac을 위한 텍스트 에디터인 Sublime Text 2에 대해 소개하고자 합니다.

사실 TextWrangler도 좋은 에디터이긴 하지만 서브라임 텍스트2Sublime Text 2도 만만치 않게 좋은 에디터입니다. 요즘은 이것을 사용하고 있어요.

공식 홈페이지

서브라임 텍스트2의 공식 홈페이지는 http://www.sublimetext.com/입니다. 이곳에서 서브라임 텍스트2를 다운로드 받을수 있어요.

라이센스

라이센스가 걱정이 되어 검색을 좀 했더니 다음과 같은 내용이 있었습니다.

Sublime Text 2 may be downloaded and evaluated for free, however a license must bepurchased for continued use. There is currently no enforced time limit for the evaluation.

http://www.sublimetext.com/2

일단 무료라는 점이지만 지속적인 사용을 하려면 구매를 하라고 되어 있네요. 구매를 하지 않으면 파일을 Save 할때마다 가끔 결제를 유도하는 윈도우가 나옵니다.

단축키(hot key)

단축키는 공식 홈페이지에서도 소개가 되어 있지만 개인적으로 자주 써볼만한 커맨드를 정리해 봤습니다.

* Command + P
접두사 #을 붙이면 파일 내부 일치하는 내용 찾기( #import )
접두사 :을 붙이면 해당 라인으로 이동( :20 )

* Command + Shift + P
명령 일람

* Command + D
현재 단어(Word) 하나를 기준으로 계속 Ctrl + D를 하면 같은 단어를 선택하고 이후 변경하면 한번에 변경된다.

* Command + Shift + F
Replace 명령, 정규식으로도 바꿀 수 있다.

* Command + Option + G
현재 단어(Word) 하나를 기준으로 같은 파일 내에 같은 단어를 전부 선택한다. 이후 변경하면 한번에 변경된다.

* Command + F
이후 Option + Enter 로 다중 선택 이후 해야할 작업을 수행한다.

* Command + [ , Command + ]
선택된 블록 들여쓰기/내어쓰기
